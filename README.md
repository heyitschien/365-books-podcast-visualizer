This repository will stay in sync with your deployed chats on [v0.app](https://v0.app).
Any changes you make to your deployed app will be automatically pushed to this repository from [v0.app](https://v0.app).

## Tech Stack

- **Framework**: Next.js 15 with the App Router, React 19, TypeScript 5
- **Styling**: Tailwind CSS 4 with PostCSS + Autoprefixer, `clsx`, `tailwind-merge`, `geist` font
- **UI Components**: Radix UI primitives (`@radix-ui/react-*`), `cmdk`, `vaul`, `sonner`, `react-resizable-panels`
- **Forms & Validation**: `react-hook-form`, `@hookform/resolvers`, `zod`
- **Data & Utilities**: `date-fns`, `recharts`, `next-themes`, `@vercel/analytics`
- **Build & Tooling**: pnpm, ESLint, TypeScript, Vercel deployment hooks

## Visual Effects & 3D Stack

- **Rendering Core**: `three`, `@react-three/fiber`, and `@react-three/drei` for scene helpers
- **Scene Instrumentation**: `r3f-perf` for performance overlays, `maath` for math helpers, `leva` for live controls
- **Native/WebGL Bridges**: Expo (`expo-gl`, `expo-asset`, `expo-file-system`) for shared shader assets
- **Animation & Feedback**: `tailwindcss-animate`, `tw-animate-css`, `sonner` to complement motion and system toasts
